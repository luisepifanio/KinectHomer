ComfyJDemo  demonstrates the ComfyJ  ability to  integrate ActiveX controls into
a Java Swing  application and  to  interact  with  them  using   the  Automation
technology. 

In order to run ComfyJDemo you should put the ComfyJ license file into the "bin"
subfolder.
